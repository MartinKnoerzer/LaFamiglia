# Vereinsrestaurant Website

Welcome to the Vereinsrestaurant website project! This project is designed to provide an online presence for a restaurant associated with a club, featuring essential information and functionalities for users.

## Project Structure

The project is organized as follows:

```
vereinsrestaurant-website
├── src
│   ├── pages
│   │   ├── index.html          # Homepage of the restaurant
│   │   ├── speisekarte.html    # Menu page listing available dishes
│   │   ├── reservierung.html    # Reservation form for users
│   │   └── kontakt.html        # Contact information page
│   ├── styles
│   │   └── main.css            # CSS styles for the website
│   ├── scripts
│   │   └── main.js             # JavaScript for interactive features
│   └── components
│       ├── header.html         # Header component with navigation
│       └── footer.html         # Footer component with additional links
├── .gitignore                  # Files and directories to ignore in version control
└── README.md                   # Project documentation
```

## Features

- **Homepage**: An inviting landing page that introduces the restaurant.
- **Menu**: A detailed menu showcasing the dishes offered, complete with descriptions.
- **Reservation**: A user-friendly form for making reservations, including options for date, time, and guest count.
- **Contact**: Easy access to contact information and a form for inquiries.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open the `index.html` file in your web browser to view the homepage.
4. Use a local server for full functionality, especially for the reservation form.
