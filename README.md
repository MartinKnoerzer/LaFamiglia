# La Famiglia 🍕

Eine moderne Website für das italienische Restaurant **La Famiglia**.

## Über das Projekt

Diese statische Website präsentiert das Restaurant La Famiglia mit:

- **Hero-Bereich** – einladendes Bild mit Slogan
- **Über uns** – Geschichte und Werte des Restaurants
- **Menü** – Spezialitäten und Preise
- **Kontakt** – Öffnungszeiten, Adresse und Reservierungsformular

## Dateien

| Datei | Beschreibung |
|-------|-------------|
| `index.html` | Haupt-Webseite |
| `styles.css` | CSS-Styling |

## Lokale Vorschau

Einfach `index.html` im Browser öffnen – keine Installation erforderlich.
