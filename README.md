# La Famiglia – Vereinsrestaurant Website

Eine einfache, responsive HTML-Website für das Vereinsrestaurant **La Famiglia**.

## Dateien

| Datei | Beschreibung |
|-------|--------------|
| `index.html` | Haupt-HTML-Datei mit allen Sektionen |
| `styles.css` | CSS-Styling (responsive, mobil-optimiert) |
| `script.js` | JavaScript für Navigation, Formular etc. |

## Inhalt der Website

- **Home** – Hero-Bereich mit Call-to-Action
- **Über uns** – Kurzbeschreibung des Restaurants
- **Speisekarte** – Vorspeisen, Hauptgerichte, Desserts
- **Reservierung** – Online-Reservierungsformular
- **Kontakt & Öffnungszeiten**

## Verwendung

Einfach `index.html` im Browser öffnen – keine Server-Installation erforderlich.
